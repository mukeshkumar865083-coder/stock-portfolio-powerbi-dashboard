Stock Portfolio Performance Dashboard

This project is a dynamic Stock Portfolio Dashboard built using Power BI to analyze investment performance, risk, and returns.

🚀 Key Features
Total Investment, Current Value & Profit/Loss KPIs
Top 5 Best Performing Stocks
Bottom 5 Loss-Making Stocks
Portfolio Allocation (Pie Chart)
Risk Level Breakdown
Stock Performance Funnel
Return % Analysis
📈 Tools Used
Excel (Data Cleaning & Preparation)
Power BI (Dashboard & Visualization)
